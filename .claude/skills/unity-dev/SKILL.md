# Unity Dev — Unity 游戏开发工具集

## 子技能列表

| 子技能 | 适用场景 | 参考文件 |
|--------|---------|---------|
| **unity-skills** | Unity Editor REST API 自动化：控制 GameObject、组件、场景、材质、预制体、灯光等（100+ 工具） | `references/unity-skills.md` |
| **game-ui-design** | 游戏 UI 设计理论：融合任天堂清晰性、沉浸式叙事界面、电竞可读性的设计原则 | `references/game-ui-design.md` |
| **animate** | 动画与微交互：评审功能并通过有目的性的动画、微交互来优化体验 | `references/animate.md` |
| **cog-config-table-designer** | COG 项目配置表设计：DataTable 规范、TXT 格式生成、资源 ID 规范 | `references/cog-config-table-designer.md` |
| **frontend-design** | 前端界面设计：创建高设计水准的 Web 组件、页面、仪表板、HTML/CSS 布局 | `references/frontend-design.md` |

## 使用流程

1. 根据用户需求类型匹配最合适的子技能
2. 读取对应的 `references/*.md` 获取详细操作指令
3. 按照指令执行任务

## 选择指南

- **Unity Editor 操作**（创建/修改 GameObject、组件、材质等） → unity-skills
- **UI 设计评审/理论指导** → game-ui-design
- **动画/过渡/微交互优化** → animate
- **新建/修改配置表** → cog-config-table-designer
- **Web 前端界面/HTML 页面** → frontend-design
