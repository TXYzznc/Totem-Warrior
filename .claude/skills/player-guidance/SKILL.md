# Player Guidance (Spatial Direction)

## 何时使用
- 设计关卡布局、玩家行进路线时
- 玩家在 playtest 中频繁迷路 / 错过关键路径
- 需要平衡「自由探索」与「叙事推进」
- 减少 HUD 标记、依赖环境引导（沉浸感优先）
- 设计隐藏路径 / 探索奖励的反引导

## 核心原则
1. **关卡是隐形的导演**：玩家以为自己在自由选择，实际上每个选择都是设计师预设的。Half-Life 2、The Last of Us 是大师课。
2. **引导分四级**：HUD 标记（最强、最破坏沉浸）→ NPC 对话提示 → 环境视觉提示 → 玩家直觉（最弱、最沉浸）。能用低级别就别用高级别。
3. **冗余引导（Redundancy）**：关键路径至少 2-3 个独立线索同时指向（光 + 声音 + 形状）——任何单一线索失效仍有备份。
4. **affordance（可供性）必须一致**：黄色 = 可攀爬，全游戏不能变。打破玩家学到的视觉语言＝惩罚。
5. **反引导也是引导**：隐藏路径用反向视觉语言（暗、隐蔽、需要回头）——但仍需「可发现」线索。

## 关键模式

### 模式 A：Sightline 引导（远处地标）
玩家天生会朝**视野中最显眼的对象**移动。利用：
- **远处的塔 / 高建筑**：BotW Hyrule Castle、Skyrim 高吼修道院——开放世界永恒地标
- **垂直元素**：高耸物体在地平线打破水平，强烈吸引目光
- **移动元素**：远处飞鸟 / 烟囱冒烟 / 旗帜飘动——动比静更吸睛
- **光柱 / 光晕**：Mirror's Edge 红色物体、Bioshock Infinite 光柱标记目标

设计法则：从玩家出生点旋转 360°，至少有 1-2 个清晰地标。Half-Life 2 Ravenholm 入口的城堡剪影是经典案例。

### 模式 B：Color & Contrast Cue
- **Mirror's Edge 红色**：所有可交互/可跑酷物体喷成红色——颜色即语言
- **Dishonored 黄色**：可攀爬边缘、可掠夺物品黄铜色调
- **The Last of Us 黄色梯子/绳子**：玩家学过一次「黄色 = 可用」终身有效
- **Uncharted 黄色把手**：纳米级一致性，全系列都是
- **冷暖对比**：暖色（橙红黄）吸引前进，冷色（蓝灰）背景退后——电影业百年技巧

**坑**：色盲玩家——颜色提示必须配合形状/对比度（Mirror's Edge 后期版本加入色盲模式重新着色）。

### 模式 C：Affordance（可供性）设计
玩家应**一眼看出**什么能做：
- **可攀爬墙面**：手印、磨损痕迹、绳索（神秘海域）
- **可破坏墙**：裂纹、不同材质、颜色异常（Zelda 系列）
- **可推动箱子**：单独摆放、与环境分离、阴影投射独立
- **可交互 NPC**：站姿、动画 idle 比路人更活跃
- **不可逾越边界**：明显的悬崖、栏杆、视觉的「死结」——而非透明墙

**Don Norman 设计法则**：affordance 应「不教自明」。如果需要文字解释「按 X 跳过这里」——设计已失败。

### 模式 D：Breadcrumbing（面包屑）
小奖励引导路径：
- **金币 / 收集物链**：Mario / Sonic 金币路径直接画出跳跃轨迹
- **物资细节**：弹药、医疗包放在正确路径的视野范围
- **环境叙事痕迹**：尸体、血迹、撕裂的纸条引向下一节点（The Last of Us）
- **微观地标**：每 30-50 米一个视觉变化（雕像、不同灯、植物），让玩家知道「还在路上」

**法则**：玩家任何时刻看向前方应能看到下一个「锚点」——锚点之间距离不超过一个房间。

### 模式 E：Lighting 引导路径
光是最强的潜意识引导：
- **明 vs 暗**：玩家天生走向光（恐惧逻辑——洞穴时代）。Resident Evil 7、Alien Isolation 用这点制造选择困境
- **冷光 = 路径，暖光 = 重要点**：Dishonored 用蓝色环境光做引导、暖光照亮关键 NPC
- **光柱标记**：神圣 / 目标场所往往加垂直光柱（Bioshock Infinite、Hades 房间出口）
- **逆光剪影**：远处地标做成逆光剪影，强烈识别（Inside、Limbo 全程靠剪影）
- **窗户与门**：透光的开口暗示「下一区域在这里」

### 模式 F：Sound Cue 引导
- **环境音渐变**：靠近瀑布听到水声、靠近战斗听到金属碰撞——Dead Space 是大师
- **NPC 远程呼喊**：Half-Life 2 Alyx 喊「这边走！」是经典
- **音乐切换**：进入新区域音乐变化（Dark Souls 篝火 / Boss 战）
- **3D 定位音**：耳机左右声道指向目标方向

注意：**音频引导不可独占**——失聪玩家需要视觉冗余。

### 模式 G：反引导（隐藏奖励）
让 5-10% 玩家发现的内容：
- **视觉打破**：墙上一块砖颜色略异、地砖图案中断——Zelda、Hollow Knight
- **回头视角**：只有从相反方向走才能看到的开口（Dark Souls）
- **跳跃信仰**：明显有「下不去」的坑，但下去有奖励（Witcher 3 宝藏地点）
- **环境矛盾**：荒野中一个孤立的木桶、室内一面突兀的画——程序生成关卡也要保留这种「人造痕迹」

**法则**：隐藏内容不该比正路重要（避免错过感）。但找到时奖励要够爽（独家 Boss、装备、剧情）。

### 模式 H：HUD vs 环境引导取舍
| 维度 | HUD 引导（minimap / waypoint） | 环境引导 |
|---|---|---|
| 学习成本 | 0 | 高 |
| 沉浸感 | 低 | 高 |
| 适合类型 | 开放世界、MMO、新手向 | 沉浸式模拟、恐怖、艺术游戏 |
| 风险 | 玩家盯着 minimap 不看世界 | 玩家迷路流失 |
| 折中方案 | 可关闭 / 简化（Witcher 3 Death March、Skyrim Compass） | 加可选探索辅助（BotW 双筒望远镜） |

**法则**：默认不要堆 HUD，而要让环境足够强；为不擅长找路的玩家提供**可选**辅助（Assassin's Creed Origins Exploration Mode）。

## 案例参考

- **Half-Life 2**：Ravenholm 章节开场——头顶灯泡 + 警报声 + 屋顶剪影 + 角色对话 = 四重冗余引导
- **Mirror's Edge**：「Runner Vision」红色系统——颜色即引导，无 minimap
- **The Last of Us**：黄色梯子、绳子、把手——15 年一致语言
- **Dishonored**：垂直关卡设计 + 黄色金属边缘 + 远处地标 = 自由路径选择
- **Dark Souls**：完全无 minimap，靠地标（防火女、不死镇钟楼）+ 环境叙事
- **Inside / Limbo**：纯剪影 + 光照，零文字，零 HUD
- **Hollow Knight**：地图分区域购买 + 远处地标声音 + 角落隐藏路径
- **BotW**：Hyrule Castle 永远在视野中 + 神庙橙色发光 + 双筒望远镜标记

## 常见坑
- **坑：玩家迷路 50% 在某节点**。原因：sightline 被遮挡 / 关键 affordance 不明显。修法：playtest 录屏，找出「玩家头转 5+ 次」的地点，加视觉锚点。
- **坑：玩家盯着 minimap 不看世界**。原因：HUD 信息过载。修法：默认关闭精确标记，改用「方向感」（罗盘上模糊指示）。
- **坑：可攀爬墙和不可攀爬墙看起来一样**。原因：affordance 不一致。修法：建立 art bible，全局应用同一规则（颜色 / 材质 / 形状）。
- **坑：色盲玩家找不到提示**。原因：纯颜色编码。修法：颜色 + 形状 / 对比度双重编码。
- **坑：玩家走完场景没看 90% 的设计**。原因：路径过于线性。修法：在主路径上摆放「钩子」回望可见的隐藏路径——激发好奇心。
- **坑：玩家不知道这是 cutscene 还是可操作**。原因：相机切换不清。修法：明确黑场 / UI 隐藏 / 操作锁定，让玩家知道当前模式。
- **坑：开放世界跑路无聊**。原因：路上没有 breadcrumb。修法：每 30-50 米一个小事件（敌人、收集、风景变化）。
- **坑：玩家错过关键剧情触发**。原因：触发器位置偏离自然路径。修法：触发器放在视线必经之处 + NPC 呼喊冗余。

## 输出形式

### 1. 关卡引导规格图（Top-Down Layout）
- 主路径（蓝色实线）
- 隐藏路径（黄色虚线）
- Sightline 锥（标注玩家在各点可见地标）
- 光照引导节点（标注光源位置 + 颜色）
- Sound cue 触发区
- Affordance 元素清单（可攀爬 / 可破坏 / 可拾取）

### 2. 引导冗余检查表
每个关键转折点 ≥ 2 个独立引导信号：
| 节点 | 视觉 | 光照 | 声音 | NPC 提示 | HUD（可选） |
|---|---|---|---|---|---|
| A → B 转折 | ✓ 远处塔 | ✓ 暖光门 | ✓ 战斗声 | - | - |

### 3. Playtest 观察清单
- 玩家在每个节点用时（>2 倍预期＝可能迷路）
- 玩家头转次数（>5 次＝信号不足）
- 玩家是否发现隐藏路径（目标发现率 5-10%）
- 玩家是否走「反方向」（如果是，反方向是否有 dead-end 引导拉回）
