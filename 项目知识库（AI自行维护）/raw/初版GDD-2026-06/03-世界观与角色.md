# 03 世界观与角色

## 一、世界观定位

### 大背景：多元末日废墟

世界因**多种灾难叠加**而崩塌，不同地区由不同末日成因主导，废墟中残存着各式各样的危险与机遇。

### 三大末日成因

| 成因 | 描述 | 留下的痕迹 |
|------|------|---------|
| 🤖 **AI 叛乱** | 高科技 AI 觉醒并反叛人类，机器人战争摧毁了文明 | 废弃工厂、失控机械、电子残骸、能量残留 |
| 👽 **外星入侵** | 异界生物入侵，带来未知科技与法则改变 | 异形巢穴、扭曲空间、外星造物、生物变异 |
| 🦠 **病毒变异** | 未知病毒席卷全球，幸存生物大量变异 | 感染区、变异生物群、毒液沼泽、生化实验残留 |

### 世界观服务的设计目的

- ✅ **支持不同地图主题**：每局可随机选定一种或多种主题，丰富视觉与玩法
- ✅ **支持不同事件池**：AI 区域有机器人 BOSS，外星区有异形巢穴
- ✅ **绑定纹身颜色系统**：每种成因对应一类元素颜料
- ✅ **支持弱剧情下的强氛围**：玩家无需理解长篇剧情，环境本身就讲故事

## 二、玩家角色

### 玩家身份：实验体 / 被改造者

在末日中，玩家扮演被某种力量改造的**实验体**——身体已被植入或修改，具备承载"纹身能量"的能力。

### 身份设定的妙处

| 设计意图 | 解释 |
|---------|------|
| 解释为何能刻纹身 | 普通人刻纹身无效果，实验体身体已被改造 |
| 解释为何身体重置 | 每局是一次"觉醒"，纹身随死亡褪去（回归实验体状态） |
| 解释为何能力多样 | 实验体本身是"画布"，承载何种能力取决于纹身 |
| 自然支持多角色 | 不同实验体（不同改造程度 / 形态）= 不同角色 |
| 视觉天然吸引 | 改造痕迹 + 纹身的组合非常有视觉冲击力 |

### 初始角色（MVP）

- **第 1 个角色**：标准实验体
  - 6 部位纹身槽全开
  - 中庸属性（基础生命、攻击、移速）
  - 默认武器：基础近战 + 基础远程
- 后续角色（DLC / 解锁）：
  - 可在不同部位有起始底纹（提供方向偏好）
  - 不同体型 / 形态（如机械化、植物化、异形化）
  - **数值平衡**，仅风格差异

## 三、世界观与系统的契合

### 颜色 ↔ 末日成因对应

| 颜色 | 元素 | 主要来源 |
|------|------|---------|
| 🔴 红 | 火焰 | 病毒变异 / 异变能量 |
| 🟡 黄 | 雷电 | AI 故障 / 机械能量 |
| 🟢 绿 | 自然 / 毒 | 病毒变异 / 生化污染 |
| 🔵 蓝 | 冰霜 | 外星造物 / 异界寒能 |
| 🟣 紫 | 异变 / 虚空 | 外星入侵 / 维度扭曲 |
| 🟨 金 | 神圣 / 黄金 | 神秘 / 古文明残留 |
| ⚪ 白 | 纯能 / 基础 | 中性 / 实验体本能 |

### 为什么这种设定优雅

- **地图主题决定可获颜料**：进 AI 区域多见黄墨、进病毒区多见红绿
- **玩家选择主题区域 = 选择 build 方向**：而非通过任意刷怪硬凑
- **环境本身在引导玩法**：不需要任何对话框告诉玩家"这里是火焰区"
- **支持季节性内容**：未来扩展新末日成因（如奥术崩坏、海洋异变）= 加新颜色

## 四、剧情立场

### 弱剧情设计

- **没有强制剧情线**，避免打断 BR 节奏
- 玩家无需理解世界观即可游玩
- 世界观通过**环境讲述**（环境叙事、物品描述、视觉风格）

### 提供剧情挖掘的入口（可选）

- 物品 / 装备的简短描述
- 纹身师 NPC 的对话片段
- 地图特殊点位的环境叙事
- 角色解锁背景故事（每个角色有简短身世）

> 剧情粒度类比《Dead Cells》《Hades》— 不强塞剧情但深入挖掘有内容。

## 五、视觉氛围（待美术风格确定后细化）

### 氛围关键词
- **废墟感**：破败建筑、断壁残垣、随处可见的灾难痕迹
- **改造感**：玩家角色身上的实验改造痕迹、机械义体可见
- **元素冲突**：火焰废墟旁是冰冻异形巢穴，强烈视觉对比
- **生命与死亡共存**：废墟中有突破而生的植物、被感染但仍在挣扎的生物

### 视觉差异化（与同类游戏对比）

| 同类游戏 | 主要美术氛围 | 本作差异化 |
|---------|------------|------------|
| Fallout | 后核辐射废土 | 我们多元末日（不止核），有外星感与异变感 |
| Borderlands | 卡通暴力 | 我们更克制，写实与抽象结合 |
| Dead Cells | 哥特城堡 | 我们后现代废墟与高科技遗迹混合 |
| Hades | 古希腊神话 | 我们科幻末日与改造主题 |
