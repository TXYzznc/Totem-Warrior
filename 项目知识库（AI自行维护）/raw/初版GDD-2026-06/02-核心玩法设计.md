# 02 核心玩法设计

## 一、视角与表现

### 摄影视角：俯视角 2.5D

- 类似《数字上帝国》《Stardew Valley》《Vampire Survivors》的鸟瞰视角
- 玩家能看到整个角色与周围地图区域
- 2.5D 提供更好的视觉层次，纹身效果可在角色身上清晰呈现
- 兼顾 PC 鼠标 + 移动端触控操作

### 美术风格

- **风格定位**：待主题深化后再确定
- **候选方向**（未决）：像素尼风 / 手绘风 / 简约几何 / 写实 2.5D
- 风格选择需服务于：① 末日废墟氛围 ② 纹身视觉清晰度 ③ 移动端性能

## 二、单局结构

### 时长与人数
- **单局长度**：15-25 分钟（中等局长）
- **单局玩家数**：50 人（含人机 AI 玩家填充）
- **队伍模式**：主推 Solo，可邀请好友共同进局组队

### 局内节奏分阶段

| 阶段 | 时长 | 关键体验 |
|------|------|---------|
| 早期 | 0-8 分钟 | 落地、探索 PVE、积累纹身资源、形成初步 build |
| 中期 | 8-18 分钟 | 圈心偏移、玩家聚集、死亡宝箱密集出现、PVP 升温 |
| 后期 | 18-25 分钟 | 圈急速收缩、决战、高稀有度刷新、终极 build 决胜 |

## 三、战斗系统

### 战斗手感四件套

1. **普攻**（武器相关）
   - 与当前装备武器绑定
   - 不同武器决定普攻风格（近战 / 远程 / 范围）

2. **蓄力普攻**（武器相关）
   - 长按触发，伤害和效果显著强化
   - 与纹身联动（如左臂蓄力相关纹身可强化此机制）

3. **多技能槽位**（局内构筑）
   - 局内通过事件 / 拾取获得
   - 多个技能可同时装备
   - 与纹身联动产生 build 差异

4. **闪避**（无敌帧 + 手感保障）
   - 提供无敌帧躲避攻击
   - 是基础生存机制，纹身可强化此机制（右腿系列）

### 武器系统（草案）
- 武器是局内核心装备，决定普攻基调
- 武器分类候选：近战（锤 / 刀 / 拳）、远程（枪 / 弓 / 投掷）、特殊（能量类）
- 武器获取：地图刷新、商人、击杀掉落、死亡宝箱
- **细节待深入**，详见 [`08-待讨论事项与下一步.md`](08-待讨论事项与下一步.md)

## 四、肉鸽元素（局内成长）

### 资源获取渠道

| 渠道 | 内容 |
|------|------|
| 🎁 地图宝箱 | 随机刷新，颜料、配方、武器、技能 |
| 💼 商人 NPC | 用金币 / 资源换装备和服务 |
| 💀 击杀掉落 | 击杀敌人（玩家或怪物）后落地的奖励 |
| 🗃 死亡宝箱 ⭐ | 死亡玩家原地生成宝箱，包含其本局全部物资 |
| 🏛 纹身师 NPC | 提供刻纹身服务，可能售卖稀有图案配方 |
| 🎪 局内事件 | 随机触发的小事件 / 试炼，奖励特殊物品 |

> ⭐ **死亡宝箱是本作的核心 PVP 诱饵设计**，将玩家死亡转化为地图热点，详见 [04-纹身系统总览](04-纹身系统总览.md) 与 [01-项目概述](01-项目概述.md)。

### 局内成长循环

```
落地 → 探索 / 战斗 → 获取颜料、配方 → 进纹身师工作室 → 刻纹身
       ↑                                                       ↓
       ←————————————————— 强化角色 ——————————————————————————←
```

## 五、大逃杀机制

### 地图

- **生成方式**：全随机生成大地图
- **地图主题**：由"末日成因"决定本局主色调（AI 废墟 / 外星侵蚀区 / 病毒变异区）
- 不同主题影响：怪物类型、颜料分布、事件池、视觉氛围

### 缩圈机制（推荐设计）

```
早期（0-8 分钟）：圈大且慢
  └─ 让玩家有时间探索 PVE、积累资源、形成初步 build

中期（8-18 分钟）：圈心随机偏移
  └─ 推动玩家聚集，死亡宝箱出现密集，PVP 冲突点爆发

后期（18 分钟+）：圈急速收缩
  └─ 强迫决战，高稀有度内容在危险区刷新，制造高风险高回报
```

**关键创新点**：
- 圈外不是纯粹的死亡区，仍会刷出**稀有怪和高级宝箱**
- 让"赌一把进圈外"成为有意义的策略选择
- 区别于传统 BR "缩圈 = 别人的麻烦" 的设计

### 人数填充策略

- 50 人局，根据实际玩家数动态填充 AI 人机
- AI 行为应尽量模拟真实玩家（拾取、躲圈、PVE、回击）
- 初期版本依赖 AI 保证体验完整，玩家基数扩大后比例自然下降

## 六、PVE + PVP 平衡

### 设计原则

- **PVE 内容**：提供资源、成长、低风险节奏空间
- **PVP 内容**：提供高强度奖励、淘汰压力、最终竞争
- **两者比例**：地图上 PVE 与 PVP 资源分布需精心调试，避免一边倒

### 避免陷阱

- ⚠️ **PVE 刷资源太快** → PVP 玩家被秒杀（建议 PVE 节奏匹配缩圈节奏）
- ⚠️ **PVE 太难** → 玩家来不及成长就被淘汰（建议 PVE 难度分级分布）
- ⚠️ **PVP 收益碾压 PVE** → 玩家全员蹲守抢人头（通过死亡宝箱稀有度上限平衡）

## 七、局外成长

### 设计原则
> **局外成长不影响局内强度，只扩展可玩内容**

### 解锁内容

| 类别 | 解锁方式 | 影响范围 |
|------|---------|---------|
| 新角色 | 完成成就 / 任务 | 提供不同身体起始底色 / 形态，**不影响数值** |
| 局内内容池扩展 | 已解锁的角色 / 进度 | 增加图案模板、颜料种类等池子内容 |
| 外观皮肤 | 局外任务 / 内购 | 纯展示，无影响 |

### 反 P2W 承诺

- 永远不通过付费让玩家获得局内数值优势
- 局外解锁的角色仅提供选择性，**不存在更强的角色**
- 不同角色应是不同**风格**，而非不同**强度**

## 八、玩家输入与操作

### PC 端（首发）
- WASD 移动
- 鼠标矋准 / 射击
- 数字键切换武器 / 技能
- 空格闪避
- E 互动 / 拾取

### 移动端（未来）
- 双虚拟摇杆（移动 + 矋准）
- 按钮：闪避、技能、互动
- UI 需适配触屏点击精度

> 移动端 UX 需在 MVP 阶段就建立性能预算，避免后期重做。
