# 04 纹身系统总览

> **本文档为纹身系统的高层概览与设计哲学**。
> 完整的 21 原子定义 + 336 种组合枚举详见 [`05-纹身系统三层原子详细设计.md`](05-纹身系统三层原子详细设计.md)。

---

## 一、为什么纹身是核心特色

| 设计目标 | 纹身系统的贡献 |
|---------|------------|
| 视觉差异化 | 玩家身体即词条展示面，远观可辨别流派与强度 |
| 玩法自由度 | 6×7×8=336 种基础组合，配合联动呈指数级深度 |
| 主题契合 | 与"实验体改造"身份完美绑定 |
| 局外不破坏平衡 | 自由装饰部分仅作美感扩展，不影响数值 |
| 替代传统装备栏 | 取消"换装备"，强化"画身体"的肉鸽体验 |

---

## 二、设计哲学：符印 + 自由装饰混合

### 双层结构

| 层 | 角色 | 决定 |
|----|------|------|
| **符印（Sigil）** | 机制层 | 提供数值效果，影响 build 强度 |
| **自由装饰** | 美感层 | 玩家自由绘制，不影响机制 |

### 为什么不二选一

- ❌ **纯自由绘制**：玩家会发现画啥效果都一样，停止认真创作
- ❌ **纯配方限定**：Meta build 趋同，所有强者长得一样
- ✅ **混合模型**：机制由符印保证（可平衡可实现），自由由装饰保证（美感与表达）

## 三、三层正交结构

### 基本结构

```
6 部位 (固定结构)
   │
   ├─ 决定: 强化哪个属性 + 何时触发
   │
   ├─ × 7 颜色 (元素派系)
   │      │
   │      ├─ 决定: 用什么元素
   │      │
   │      └─ × N 图案 (行为触发)
   │             │
   │             └─ 决定: 怎么触发 / 表现
   │
   └─→ 6 × 7 × N 个独立组合 × 跨层联动 = 海量构筑空间
```

### 各层定义速览

#### 6 个部位（固定）

| 部位 | 触发事件 | 强化属性 |
|------|---------|---------|
| 🧠 脑袋 | 暴击命中时 | 暴击率 / 暴击伤害 |
| 🫀 躯干 | 受伤时 | 生命上限 / 防御 |
| 💪 左臂 | 普攻命中时 | 主武器伤害 / 攻速 |
| 🤚 右臂 | 技能释放时 | 技能强度 / 冷却 |
| 🦵 左腿 | 持续移动时 | 移动速度 / 加速 |
| 🦶 右腿 | 闪避成功时 | 闪避帧 / 闪避距离 |

#### 7 种颜色（元素派系）

| 颜色 | 元素 | 主要行为 |
|------|------|---------|
| 🔴 红 | 火焰 | 燃烧、爆裂、增伤 |

| 🔵 蓝 | 冰霜 | 减速、控制、命中 |
| 🟣 紫 | 异变 | 随机状态、混乱 |
| 🟨 金 | 神圣 | 必发、奇迹、闪避奖励 |

## 五、联动机制（深度来源）

联动**不是新效果**，而是**已有效果的后处理倍率**：

```
最终效果 = 基础效果(部位+颜色+图案)
        × 同色共鸣倍率
        × 同图案呼应倍率
        × 对称构型倍率
        × 颜色反应倍率
```

### A. 颜色共鸣（同色聚集）

| 同色部位数 | 效果 |
|-----------|------|
| 3 部位同色 | 该色元素伤害 +20% |
| 4-5 部位同色 | 元素伤害 +35% + 减少敌人抗性 |
| 6 部位全同色 | 进入元素化身形态（高风险高回报） |

### B. 图案呼应（同图案聚集）
- 多部位同图案 → 该图案触发效果叠加 / 增强
- 例：3 个"闪电"图案 → 链式跳跃数 +2

### C. 对称构型（左右对称）
- 左臂 + 右臂同图案同色 → "双持协同"（武器伤害 +X%）
- 左腿 + 右腿同图案同色 → "步法精通"（移动技能进化）

### D. 颜色反应（跨色组合）

| 组合 | 反应 | 效果 |
|------|------|------|
| 🔴 + 🟡 | 爆裂 | 火雷连锁触发额外爆炸 |
| 🔵 + 🟢 | 寒毒 | 冰冻 + 毒持续叠加，无法清除 |
| 🟣 + 🟨 | 混乱奇迹 | 状态效果完全随机化，可能极强 |
| ⚪ + 任意 | 基础强化 | 白色作为"放大器"，提升相邻颜色效果 |

### E. 全身设计成就（仅局外）

- 全身覆盖率 ≥ 80% → 局外奖励"画师勋章"（解锁新画笔）
- 6 色全用 → 解锁"彩虹之子"称号
- 仅用单色 → 解锁该色专属皮肤

> 成就**不影响局内强度**，仅扩展外观与展示。

## 六、自由装饰部分（美感扩展）

### 作用范围
- 玩家在符印周围 / 之间自由绘制的图案
- 完全不影响机制效果
- 提供视觉表达与个性化

### 软性奖励驱动

| 机制 | 奖励 |
|------|------|
| 连接奖励 | 用线条把多个符印连起来，每条连接线提供微小协同 |
| 完成度评分 | 涂鸦覆盖率 / 配色和谐度 → 局外解锁素材 |
| 玩家展示 | 死亡回放、击杀镜头展示完整身体纹身 |
| 大厅画廊 | 好友 / 全服查看他人设计，社交传播 |

## 七、纹身的局内获取流程

### 资源类别

| 资源 | 用途 | 稀有度 |
|------|------|--------|
| 颜料瓶 | 每种颜色独立资源，刻纹身消耗 | 易得（每局可凑 4-5 色） |
| 图案配方 | 解锁可使用的图案模板 | 较稀有（每局解锁 2-3 个新图案） |
| 纹身师服务 | 提供刻纹身的工作站 | 地图固定刷新，玩家热点 |
| 死亡拓本 | 从死亡玩家宝箱中获得他人纹身片段 | 高价值，PVP 驱动力 |

### 刻纹身流程

```
1. 拾取所需颜料 + 配方
2. 找到纹身师 NPC 或携带式工作站
3. 进入纹身界面：选择部位 + 配方 + 颜色
4. 等待绘制时间（脆弱时刻 - 风险设计）
5. 完成后效果立即生效
6. 可在符印周围自由装饰（不消耗时间，纯美感）
```

### 风险设计：纹身脆弱时刻

刻纹身需要**原地停留几秒**（类似 Tarkov 包扎），形成关键的风险决策点：
- **野外刻**：节省时间，但可能被偷袭
- **据点刻**：安全但需要回到城镇 / 工作室

## 八、纹身师 NPC（地图战略点）

### 设定
- 地图上随机刷新若干**纹身师工作室**
- 每个工作室有不同稀有度的图案模板和颜料库存
- 是天然的玩家**聚集冲突点**

### 战略意义
- 比传统 BR 的"机场 / 军事基地"更主题化
- 提供 PVE（拿货）和 PVP（抢点）双重玩法
- 工作室有 NPC 守卫，进入需要先击败

## 九、平衡风险与对策

### 同质化风险
- **风险**：同色 + 同图案在不同部位效果相似，玩家感知不到差异
- **对策**：通过触发时机和数值差异化（如脑袋是高暴击爆发，左臂是稳定输出）

### 白色过强风险
- **风险**：白色提供稳定通用增益，可能成为版本最优解
- **对策**：白色基础值故意做低，强迫追求元素特性才能上限突破

### 兽形美术资源压力
- **风险**：每种颜色 × 6 部位的兽形需要独特美术
- **对策**：兽形限定高稀有度，限制刷新，可大幅减少美术工作量

### 数值平衡
- **风险**：336 种组合的相对强度难以全部调平
- **对策**：通过数据驱动架构，运行时按规则计算，无需逐一手调

详细数值平衡公式 → 待讨论（见 [`08`](08-待讨论事项与下一步.md)）

## 十、UX 设计要点（俯视角下的画板交互）

### 关键设计
- 进入纹身师工作室时弹出**专用画板界面**
- 显示角色全身立绘 + 部位选择 + 配方列表
- 简化的画笔工具（直线、曲线、点缀、花纹刷）
- 自动吸附到部位区域
- 实时预览效果（视觉 + 数值变化）

### 跨平台考虑
- PC：鼠标点击 + 拖动绘制
- 移动端：触屏点击 + 滑动绘制（触屏在画画上反而有优势）

### 友好性原则
- 简化的预设笔刷，不要做自由像素笔（开发量爆炸）
- 一键应用配方，复杂装饰可选不必做
- 提供"快速 build"模板（适合新手或快节奏玩家）

## 十一、扩展方向（未来 DLC / 季节）

| 扩展项 | 影响 |
|--------|------|
| 新图案 | 加 1 个 ScriptableObject + 1 个 EffectShape 类 → 自动获得 6×7=42 种新组合 |
| 新颜色 | 加 1 个 ScriptableObject + 1 个 ElementBehavior（如有需要）→ 自动获得 6×8=48 种新组合 |
| 新部位 | 罕见情况（如背部新部位） → 自动获得 7×8=56 种新组合 |
| 新联动 | 加一个小函数（约 20-40 行）→ 影响所有现有 build |
| 季节配色 | 推出"主题色"皮肤包，限定外观，不影响平衡 |

数据驱动架构保证：**新增内容是配置工作，不是代码重构**。
