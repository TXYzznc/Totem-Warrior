# 初版 GDD 设计文档（2026-06 baseline）

## 一、背景

项目立项阶段（2026-06-22 ~ 06-23）由用户 + AI 协同产出的初版 GDD，共 9 份文档，约 95KB。这是当前所有后续开发（纹身系统重构、UI 规划等）的**底层设计 baseline**。

原始路径：`开发文档/`（已迁移到 `raw/初版GDD-2026-06/`，保持原编号与结构）。

## 二、决策摘要

### 2.1 玩法定位

| 维度 | 决策 |
|---|---|
| 类型 | **Roguelike + Battle Royale** 融合 |
| 视角 | 俯视角 2.5D |
| 单局时长 | 15-25 分钟 |
| 单局人数 | 50（含 AI 玩家填补） |
| 队伍模式 | Solo 主推（支持邀请好友） |
| 局外成长 | **仅解锁新角色，不影响局内强度** |
| 缩圈机制 | 随机圈心 + 分阶段双速 |

### 2.2 核心特色：纹身构筑系统

- **三层正交结构**：6 部位 × 7 颜色 × 8 图案 = 336 种基础组合
- **混合模型**：符印（机制层，保平衡）+ 自由装饰（美感层，玩家表达）
- **合成公式**：`效果 = WHEN(部位触发) × HOW(图案形状) × WHAT(颜色元素) × SCALE(部位属性)`
- **联动机制**：同色共鸣 / 同图案呼应 / 对称构型 / 颜色反应
- **运行时架构**：21 个 ScriptableObject 原子 + 事件总线 + 效果模块在运行时合成（详见 [07-技术架构.md](../raw/初版GDD-2026-06/07-技术架构.md)）

### 2.3 世界观

- 多元末日（AI 叛乱 / 外星入侵 / 病毒变异）共存
- 玩家身份：实验体 / 被改造者 → 与纹身改造痕迹绑定
- 颜色（元素）与末日成因绑定，让世界观服务于机制

### 2.4 联机模式三层架构

| 层 | 内容 | 推进顺序 |
|---|---|---|
| Layer 1 | 标准 50 人 BR（Solo 主推） | 第三阶段 |
| Layer 2 | 2-4 人合作 PVE 副本（局外解锁素材） | 第二阶段 |
| Layer 3 | 自定义私房（合作/对抗/混合规则） | 第二阶段后 |

**MVP**：先单机原型 → 再 PVE 联机 → 再私房 → 再大型 BR。

### 2.5 技术栈

- 引擎 Unity 6（已定）
- 联机框架候选 Mirror / FishNet（前期）→ Photon Fusion 2（后期）
- 数据驱动 ScriptableObject + Inspector
- 资源管理 Addressables

## 三、被否定的备选

1. **纯自由绘制纹身**：玩家发现画啥都一样后会停止认真创作 → 改为符印+自由装饰混合
2. **纯配方限定纹身**：所有 Meta build 趋同 → 同上
3. **局外数值养成**：违反"局外不影响局内"原则，会形成 P2W 与新老玩家壁垒 → 局外仅解锁新角色
4. **传统装备栏**：与"身体即词条展示面"的视觉差异化目标冲突 → 用纹身替代装备栏
5. **资源带出（Tarkov 风）**：与肉鸽公平起点冲突 → 死亡宝箱仅局内流通
