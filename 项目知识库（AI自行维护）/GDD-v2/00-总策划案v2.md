## 一、一句话核心概念

> **末日废墟里，10-15 分钟一局，把颜料涂上身体亲手刻出 build，与 49 个对手抢光这片地图上的颜料与配方。**

> 类型仍是肉鸽 BR；v2.1 把节奏压短至 10-15 min，让"再来一局"成本足够低，也让"刻纹身"从 v2 的局间收尾动作变成局内**主动决策**。

## 二、设计基本盘（v2.1 锁死表）

| 维度 | v2.0 | v2.1 锁死 |
|------|------|------|
| 类型 | Roguelike + BR | **不变**（[v1 §设计基本盘](../raw/初版GDD-2026-06/01-项目概述.md#二设计基本盘)） |
| 视角 | 俯视角 2.5D | **不变** |
| **单局时长** | 20 分钟基准（8/10/2） | **10-15 分钟**，三段节奏 4/8/2（探宝 / 中段交火 / 决赛圈）|
| **单局人数** | 50 actor | **50 actor，1 玩家 + 20 智能 AI + 29 轻量 AI** |
| 联机形态 | 伪联机 | **不变**：单机起步 + AI 顶席，架构按真联机零改动设计 |
| **AI 完备度** | 8-10 智能 + 40 轻量 | **20 智能 + 29 轻量 + 1 玩家**：智能 AI 完整对齐真人（build / 探宝 / 刻纹身 / PVP）；轻量 AI 仅巡逻+还击 |
| **技能槽** | 3 槽 | **2 槽**（主 / 副）|
| 纹身门槛 | 仅纹身师可刻 | **玩家自纹身**：颜料 + 50 金币 + 读条 3-8 秒（部位决定时长，非战斗状态）|
| **纹身师角色** | 强化门槛 | **词缀附魔工**：在已有纹身上加词缀（暴击 / 吸血 / 减伤等）|
| **颜料分档** | 不分档 | **三档**：常见（红 / 黄 / 绿 / 蓝）/ 稀有（紫 / 金）/ 传说（白） |
| **配方来源** | 仅纹身师卖 | **四路**：局外解锁 + Boss 掉落 + 宝箱事件 + 死亡玩家拓本 |
| **死亡宝箱内容** | 颜料为主 | **半颜料 + 半配方拓本**（拓本本局可用，不入局外仓库）|
| **圈外刷稀有** | 有 | **取消**（与"缩圈逼战"诉求冲突）|
| **局外解锁** | 仅角色 | **三类**：新配方 + 新角色 + 装饰/衔号/画廊（**取消初始库存机制**，每局从零起）|
| **商人卖什么** | 配方 | **颜料 + 武器 + 技能 + 消耗品**（不卖配方）|
| **初始配方** | 未定 | **直线 + 圆环** 两个免费图案，覆盖所有部位 |
| **不死不杀** | 5-6 纹身/局 | **4-5 部位完成度**（佛系玩家也能完整体验 build 闭环）|
| **结算分享** | 仅结算页 | **结算页 + Run 重现**（短视频/GIF 自动导出，鼓励 UGC）|
| **配音** | 无 | **关键 NPC 轻微配音**（纹身师 / 商人 / Boss 入场，单句台词）|
| **美术风格** | 未定 | **似 Hades 精致 2.5D**（手绘描边 + 高对比光影 + 角色辨识度优先）|
| **UI 技术栈** | UI Toolkit | **UGUI + 轻量 MVP**（团队熟悉度优先，去除 USS 学习成本）|
| **主角设定** | 仅实验体 | **异能者**（实验体 / 游侠 双兼容，世界观可向两个方向延展）|
| 文档颗粒度 | 两层 | **不变**：15 系统 GDD + 16 模块详设 + 1 CONTRACT |
| 引擎 | Unity 6.3 LTS | **不变** + 自研 IGameModule 框架 |
| 首发平台 | PC | **不变**，移动端走包体预算 |
| 队伍模式 | 仅 Solo | **不变** |
| 本期边界 | 先文档后代码 | **不变**，`Assets/Scripts/` 零改动 |

## 三、核心差异化 USP（4 项）

| # | USP | 一句话 why |
|---|-----|------------|
| 1 | **纹身构筑（玩家亲手刻）** | 身体即 build 展示面——颜料 + 50 金币 + 读条 3-8 秒，在掩体后给自己刻一笔。`6 部位 × 7 颜色 × 8 图案 = 336` 组合 + 纹身师词缀附魔，把"换 build"变成必须在战场上抢出时间窗的高风险决策 |
| 2 | **死亡宝箱（半颜料 + 半配方）** | 死亡玩家原地落宝，里面一半是颜料、一半是配方拓本（本局可用）——杀强者既补充资源也偷流派，自然制造 PVP 热点 |
| 3 | **多元末日** | AI 叛乱 / 外星 / 病毒三主题同游戏池，颜色绑定末日成因，世界观服务机制（异能者主角同时兼容实验体 / 游侠两条叙事线）|
| 4 | **伪联机分级 AI（20+29+1）** | 真人玩家 + 20 智能 AI（与真人共用 `IPlayerController`，会刻纹身 / 抢死亡宝箱 / 找纹身师附魔 / PVP）+ 29 轻量 AI（巡逻+还击）填满 50 席。**架构上为真联机零改动**——未来只是把部分 `BotPlayerController` 换成 `NetworkPlayerController` |

USP 4 是相较 v1 的**最大升级**：v1 把"联机"作为前提，v2.1 把它降级为"接口契约"——伪联机本身就是完整产品形态。20 智能 AI 比例较 v2 的 8-10 翻倍，**确保单局 10-15 分钟里玩家碰到的对手大概率是"会刻 build 的对手"**，撑得起"杀强者"的设计承诺。

## 四、玩家旅程图（FTUE：第 1 局 → 第 10 局，单局 10-15 min）

> 描述玩家应**学会什么 / 达成什么 / 感受到什么**驱动 ta 进下一局。三段节奏：探宝（0-4 min）→ 中段交火（4-12 min）→ 决赛圈（12-15 min）。

| 局 | 应学会 | 应达成 | "再来一局"驱动 |
|---|---|---|---|
| **1** | WASD / 鼠标瞄 / 闪避 / 拾取 / `E` 交互 / 2 技能槽 | 活到 4 分钟，用初始**直线**配方 + 1 罐常见颜料刻第 1 个部位 | "原来在战斗中也能刻纹身——我想试别的颜色" |
| **2** | 颜料三档（常见 / 稀有 / 传说）+ 颜色 buff 差异 | 凑齐 1 罐稀有颜料（紫或金）刻在显眼部位 | "稀有颜料属性高一截——下一局我专门蹲稀有刷新点" |
| **3** | 死亡宝箱 = 颜料 + 配方拓本 | 主动捡 1 个死亡宝箱，用拓本临时解锁 1 个没见过的图案 | "刚才那哥们颜色组合好猛——杀他既补血又偷流派" |
| **4** | 缩圈节奏 / 4-5 部位完成度门槛 | 在中段交火期刻完 4 个部位，体验完整 build 闭环 | "我差点活进决赛圈——下次再试一次" |
| **5** | 纹身师 = 词缀附魔工 | 找纹身师在已有纹身上加 1 个词缀（暴击 / 吸血）| "原来纹身刻完后还能进一步强化" |
| **6** | 20 智能 AI 也在抢宝 / 也在刻纹身 / 也在找纹身师 | 第一次和智能 AI 单挑赢一次，捡到 ta 的死亡宝箱 | "AI 这么会玩，我能更会玩" |
| **7** | 末日主题对颜料池的影响 | 体验 1 个非默认主题（如外星），传说白色颜料更稀有 | "外星主题红色少绿色多——build 策略要换" |
| **8** | 商人 = 颜料 / 武器 / 技能 / 消耗品（不卖配方）| 用金币从商人买 1 罐传说颜料 + 1 件武器换装 | "下次想试图案 8 + 颜色 7 的极限堆叠" |
| **9** | Boss 房 = 唯一稀有配方源（已删除圈外刷稀有）| 在决赛圈前打过 1 次 Boss，掉 1 个新配方拓本 | "Boss 掉的图案我没见过——下次想专程打 Boss" |
| **10** | 系统认知闭环 + Run 重现导出 | 完成一次"流派计划 → 局内执行 → 决赛圈印证"，结算页导出 Run 重现 GIF | "我下一局想再玩同一个流派，再优化 1%；也想把这一局发给朋友" — **进入留存通道 + UGC 通道** |

> **心智演化主轴**：1-3 局 = 学规则；4-6 局 = 学策略；7-10 局 = 学**自己**的偏好。第 10 局后玩家从"在玩游戏"变成"在演自己的 build"——这是从动作肉鸽过渡到构筑肉鸽的核心钩子。

> **设计哲学**：每局必给 1 个**没见过的瞬间**（新颜料 / 新图案 / 新词缀 / AI 新流派 / 新主题），用未知性对冲单局重复感。10-15 min 单局压缩 + Run 重现共同支撑"再来一局"的低成本心智。

## 七、架构契约（CONTRACT）

> 全局事件总表 + 模块依赖图 + `IPlayerController` 抽象 + 50 actor 性能预算。
> **B-3 / B-4 阶段所有 Agent 必须按本契约写文档，不得擅自改签名。**

→ [openspec/changes/05-gdd-v2-full-design-docs/CONTRACT.md](../../openspec/changes/05-gdd-v2-full-design-docs/CONTRACT.md)

关键锁死项摘要：
- 所有战斗事件加 `Actor` 字段（v1 仅假定玩家发起，v2 50 actor 同场必须区分）
- `IPlayerController` 4 种实现：`Human` / `SmartBot` / `LightBot` / `NetworkReplay`（前 3 个本期实现，第 4 个本期占位）
- 50 actor 性能预算（v2.1 配比 1+20+29）：视野内智能 AI 60Hz / 视野外智能 AI 2Hz / 视野内轻量 5Hz / 视野外轻量 0.5Hz；20 智能 AI 比 v2 8-10 翻倍，CONTRACT 中性能预算需重新核算
- 智能 AI 与玩家**共用** `TattooModule.Equip` + `TattooModule.SelfTattoo`（v2.1 新增玩家自纹身 API） + `NPCModule.RequestAffix`（找纹身师附魔）—— AI 数据通路与玩家一致

## 与 v1/v2 的关键差异（备查）

| 项 | v1 | v2 | v2.1 | 推翻理由 |
|---|---|---|---|---|
| 联机 | Mirror / FishNet MVP | 伪联机 | **不变** | Phase A 共识 #2 |
| AI 配比 | 一句"模拟真实玩家" | 8-10 智能 + 40 轻量 | **20 智能 + 29 轻量 + 1 玩家** | v2.1 grill #11：10-15 min 短局里，玩家碰到的对手必须大概率"会刻 build"，才撑得起"杀强者"诉求 |
| 单局时长 | 15-25 min | 20 min（8/10/2）| **10-15 min（4/8/2）** | grill #8：再来一局心智成本必须更低；纹身从局间收尾变成局内决策也需要更紧凑节奏 |
| 纹身门槛 | 纹身师 | 纹身师=强化门槛 | **玩家自纹身 + 纹身师=词缀附魔工** | grill #2/#3：v2 的"必须找 NPC 才能刻"在 10-15 min 短局里是流程瓶颈；改为玩家自纹身+纹身师二级附魔 |
| 技能槽 | 3 | 3 | **2** | grill #9：与 10-15 min 节奏匹配，降低首局学习负担 |
| 死亡宝箱 | 颜料为主 | 颜料为主 | **半颜料+半配方拓本** | grill #7：让"杀强者"同时获得资源与流派情报，强化 PVP 动机 |
| UI 技术栈 | — | UI Toolkit | **UGUI + 轻量 MVP** | grill #1：团队熟悉度优先，去除 USS 学习成本 |
| 主角 | 实验体 | 实验体 | **异能者（双兼容）** | grill #14：保留世界观弹性，不锁死单一叙事线 |

所有 v2.1 差异均源于 16 轮 grill 24 项决策；未列入上表的变更（美术风格 / 初始配方 / 商人售卖 / Run 重现 / 配音）已在 §二 设计基本盘表完整记录。
