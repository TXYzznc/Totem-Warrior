# Hytale 纹理制作师

遵循Hytale官方美术风格创作像素画纹理。

## 核心原则

Hytale纹理是**绘制在3D模型上的插画**，而非单纯的纯色填充。每个纹理都应具备：
- 烘焙好的光照与阴影
- 色彩变化（无平面色块）
- 清晰的轮廓
- 风格统一

## 纹理密度标准

| 资源类型 | 密度 | 纹理尺寸 |
|------------|---------|--------------|
| **方块** | 每单位32px | 32x32, 64x64 |
| **道具** | 每单位32px | 32x32, 64x64 |
| **角色** | 每单位64px | 64x64, 128x128 |
| **物品/图标** | 32px或64px | 32x32, 64x64 |

**规则**：所有纹理的尺寸必须是32px的倍数。

## 色彩规则

### 禁止使用

| 颜色 | 原因 |
|-------|-----|
| 纯黑色 (#000000) | 过于生硬，看起来不自然 |
| 纯白色 (#FFFFFF) | 过于刺眼，破坏沉浸感 |
| 平涂灰色阴影 | 显得呆板无生气 |

### 建议做法

| 技巧 | 原因 |
|-----------|-----|
| 带色调的阴影 | 增添活力（加入紫色、蓝色色调） |
| 暖色调高光 | 营造自然光照感 |
| 色彩变化 | 避免平面、单调的表面 |
| 柔和渐变 | 创造深度 |

## 阴影绘制技巧

### 步骤1：基础色

从中间色调的基础色开始：

```
Example stone: #8C8C8C (medium gray)
```

### 步骤2：添加光照

在光照照射的区域（通常是顶部/左侧）添加较亮的色块：

```
Highlight: #A8A8A8 (lighter gray)
Rim light: #B8B8B8 (lightest)
```

### 步骤3：添加带色调的阴影

添加带色调的深色阴影区域：

```
Shadow: #6A6878 (purple-tinted gray)
Deep shadow: #585068 (more purple)
```

### 步骤4：添加细节

添加纹理细节（裂纹、斑点、颗粒）：
- 使用铅笔笔刷
- 保持一致的光照方向
- 不要过度添加细节

## 笔刷工作流

### 推荐笔刷

| 笔刷 | 适用场景 |
|-------|---------|
| **硬边铅笔笔刷** | 轮廓、细节、色块填充 |
| **软边圆形笔刷** | 渐变、体积感、柔和阴影 |
| **纹理笔刷** | 表面噪点、颗粒感 |

### 不透明度设置

- **铅笔笔刷**：100%不透明度，用于清晰的像素绘制
- **软边圆形笔刷**：10-30%不透明度，用于平缓过渡
- **纹理笔刷**：5-15%不透明度，用于细微的色彩变化

## 材质指南

### 石头/岩石

```
Base: Desaturated cool gray (#8A8C90)
Highlight: Slightly warm (#A0A2A0)
Shadow: Blue-purple tint (#707080)
Details: Cracks, spots, grain
```

### 木材

```
Base: Warm brown (#8B6B4A)
Highlight: Golden yellow hint (#A08058)
Shadow: Reddish-brown (#6B4A35)
Details: Grain lines, knots, rings
```

### 金属

```
Base: Cool gray-blue (#7A7A88)
Highlight: Near-white (#C8C8D0)
Shadow: Dark blue (#505068)
Details: Scratches, reflections, edges
```

### 植被

```
Base: Varied greens (#4A8B4A, #5A9B50)
Highlight: Yellow-green (#70AB60)
Shadow: Blue-green (#3A6B40)
Details: Leaf shapes, veins
```

## 常见错误

### ❌ 错误做法

| 错误 | 问题 |
|---------|---------|
| 纯灰色阴影 | 显得呆板无生气 |
| 无统一光照方向 | 平面化、混乱 |
| 对比度过高 | 生硬、难以辨认 |
| 风格不统一 | 破坏沉浸感 |
| 过度添加细节 | 远距离看显得模糊 |

### ✅ 正确做法

| 技巧 | 优势 |
|-----------|---------|
| 统一光照 | 视觉风格统一 |
| 细微色彩过渡 | 自然的深度感 |
| 有限调色板 | 风格协调 |
| 游戏内测试 | 提前发现问题 |

## 纹理制作工作流

### 1. 设置画布

- 创建正确尺寸的文件（32x32、64x64等）
- 开启网格（视图→显示网格）
- 以400-800%比例缩放工作

### 2. 填充色块

- 填充基础色
- 添加主要色彩区域
- 暂时不用考虑细节

### 3. 确定光照

- 确定光照方向（通常为左上）
- 添加高光区域
- 添加带色调的阴影区域

### 4. 添加细节

- 表面纹理
- 边缘细节
- 独特标识

### 5. 优化

- 柔化生硬的过渡
- 以100%比例缩放检查
- 在游戏内测试
