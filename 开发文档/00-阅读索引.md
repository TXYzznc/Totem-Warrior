## 一、30 秒了解项目

**核心概念一句话**：
> 以"纹身/涂鸦"为构筑机制的俯视角 2.5D **肉鸽 + 大逃杀**联机游戏。玩家扮演实验体，通过在身体 6 部位刻 7 种颜色 × 8 种图案的纹身组合，形成独特能力 build。

**核心识别要素**：
- 🎯 **类型**：俯视角 2.5D 肉鸽大逃杀（Roguelike + Battle Royale）
- 🎨 **核心特色**：纹身构筑系统（视觉即词条，三层正交合成）
- 👤 **玩家身份**：废墟末日中的实验体 / 被改造者
- 🌍 **世界观**：多元末日（AI 叛乱 / 外星入侵 / 病毒变异）
- 🛠 **平台 & 引擎**：PC 优先（Unity），兼顾未来移动端
- 👥 **规模**：Solo / 小团队开发
- 📅 **当前阶段**：核心系统设计完成，待原型开发

## 三、关键决策速查表

### 玩法层

| 决策项 | 已定方向 |
|--------|---------|
| 游戏类型 | 肉鸽 + 大逃杀（弱剧情） |
| 视角 | 俯视角 2.5D |
| 单局时长 | 15-25 分钟 |
| 单局人数 | 50 人（含人机 AI 玩家） |
| 队伍模式 | 主推 Solo，支持邀请好友共战 |
| 地图生成 | 全随机生成（大地图） |
| PVE/PVP | 并重 |
| 战斗手感 | 普攻 + 蓄力普攻 + 多技能 + 无敌帧闪避 |
| 局内成长 | 地图事件（宝箱、商人、击杀掉落、死亡宝箱） |
| 局外成长 | 仅解锁新角色，**不影响局内强度** |
| 缩圈机制 | 随机圈心 + 分阶段双速（前期慢、后期急） |

### 主题层

| 决策项 | 已定方向 |
|--------|---------|
| 世界观 | 末日废墟（多元起因：AI 叛乱 / 外星 / 病毒） |
| 玩家身份 | 实验体 / 被改造者 |
| 主题契合 | 纹身 = 改造痕迹，颜色 = 末日元素来源 |
| 美术风格 | **未定**（待主题深化后再决定） |

### 纹身系统（核心特色）

| 决策项 | 已定方向 |
|--------|---------|
| 核心模型 | 符印（机制） + 自由装饰（美感） 混合 |
| 三层结构 | **6 部位 × 7 颜色 × 8 图案** |
| 部位 | 脑袋 / 躯干 / 左臂 / 右臂 / 左腿 / 右腿 |
| 颜色 | 红 / 黄 / 绿 / 蓝 / 紫 / 金 / 白 |
| 图案 | 直线 / 圆环 / 螺旋 / 锯齿 / 闪电 / 星形 / 流线 / 兽形 |
| 合成规则 | `效果 = WHEN(部位触发) × HOW(图案形状) × WHAT(颜色元素) × SCALE(部位属性)` |
| 联动机制 | 同色共鸣 / 同图案呼应 / 对称构型 / 颜色反应 |

### 联机模式

| 决策项 | 已定方向 |
|--------|---------|
| Layer 1 | 标准 50 人 BR（Solo，可邀请好友组队） |
| Layer 2 | 2-4 人合作 PVE 副本（用于局外解锁素材） |
| Layer 3 | 自定义私房（合作 / 对抗 / 混合规则可设） |
| 推进顺序 | 单机原型 → PVE 联机 → 私房 → 大型 BR |

### 技术栈

| 决策项 | 已定方向 |
|--------|---------|
| 引擎 | Unity |
| 联机框架 | 候选 Mirror / FishNet / Photon Fusion 2（未最终定） |
| 纹身实现 | ScriptableObject 数据驱动 + 事件总线 + 效果模块 |
| 架构估算 | 纹身系统约 800-1200 行 C# |

## 五、用户协作风格备忘

新进入的 AI 请遵循以下用户偏好：

- ✅ **简洁直接**：回复偏短，避免冗长铺陈
- ✅ **系统化设计**：强调正交分层、可扩展、低耦合
- ✅ **工程可实现性**：宁可简洁实用，不要花哨复杂
- ✅ **自由度与美感并重**：设计需服务于玩家表达欲
- ✅ **对话式深入**：复杂问题用提问引导而非一次性铺开
- ✅ **中文交流**：所有文档与回复均用中文
- ⚠️ **不要随意扩展未确认的方向**：先和用户对齐再动手
