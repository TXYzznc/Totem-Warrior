# Modules — 通用模块

> 框架提供的可选通用模块。这里不放任何项目语义或具体业务模块。

## 模块清单

| 模块 | Category | 职责 |
|---|---|---|
| `Scene/` | 0 启动基础层 | Unity 场景加载、卸载、进度事件 |
| `Flow/` | 2 应用协调层 | 通用流程注册、切换、取消和日志 |

## 使用规则

- 框架模块只能提供通用机制，不定义具体游戏状态、场景类型或业务规则
- 项目自定义模块应放在项目自己的代码目录中，再按需注册到 `ModuleRunner`
- 每个模块文件夹应有 `README.md` 和必要的 `Events/` 子目录
