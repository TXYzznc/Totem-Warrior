## 经验回填（已更新到 SKILL）

| 发现 | 处理 |
|---|---|
| Codex imagegen 内部用 chroma-key（绿色 #02f805）模拟透明，输出再做 remove_chroma_key + resize | 已在最终 PNG 验证 alpha 通道有效（四角 alpha=0），无需 Claude 端额外处理 |
| ImageCut 的 label 序号≠row-major 顺序 | 必须按 bbox 中心 row-bucketize（y/(canvas_h/ROWS)）+ 行内按 x 排序 |
| 同一行 icon 的 y 中心相差 2-5 像素（imagegen 自由布局） | 直接 (y,x) 排序会跨行错位 → row-bucketize 修复 |
| Codex 自动调用 PowerShell 做后处理（resize、alpha 验证、target 复制） | 沙箱 workspace-write 模式下没问题；命令模板里只需要给出"落盘路径 + grid 规格"，其他交给 imagegen |
