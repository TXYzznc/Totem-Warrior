## 表 4：视觉风格约束

> 对标 GDD-v2.1 §设计基本盘：**似 Hades 精致 2.5D**（手绘描边 + 高对比光影 + 角色辨识度优先）

| 维度 | 约束 |
|---|---|
| **配色方案** | 主色：深蓝/紫黑（背景） / 高饱和绿/金（重点）；文字：白色（主）/ 浅灰（辅）；强调：红（警告）/ 黄（重要） |
| **字体** | 中文：优雅衬线体或黑体；英文：高易读sans-serif；数字：等宽（确保对齐） |
| **描边/光影** | 所有 UI 元素：1-2px 深色描边；卡片：微投影 / 内凹阴影突出层级 |
| **动画** | 按钮反馈：0.15s 缩放；过渡：0.3-0.5s DOTween 缓动；进度条：线性或ease-out |
| **图标/资源** | 纹身部位图标：6 个（头/胸/腰/左臂/右臂/腿），辨识度高 |颜料样本：6-9 个（对应颜色档次）|图案缩略图：2-8 个（初始直线+圆环，后续 Boss 掉落） |

## SelfTattooForm 详细需求

### 视觉流程

```
玩家按 Tab
  ↓
SelfTattooForm 淡入（全屏覆盖，Sort Order 20）
  ├─ 顶部：标题"纹身工作台"
  ├─ 左侧：6 个部位按钮（头 / 胸 / 腰 / 左臂 / 右臂 / 腿）
  │  └─ 鼠标 hover 时高亮
  ├─ 中上：颜料库存网格（3 列 × 3 行）
  │  ├─ 每个格子：颜料块 + 数字
  │  └─ 0 数量时灰显禁用
  ├─ 中下：图案库选择（2 列 × 2-4 行，根据解锁数）
  │  ├─ 初始解锁：直线 + 圆环
  │  └─ 未解锁：锁图标 + "需 Boss 掉落或宝箱"
  ├─ 右侧：实时预览窗（显示选中部位 + 颜料 + 图案的效果）
  │  └─ 更新频率：实时（选颜料/图案时刷新）
  ├─ 底部：
  │  ├─ "读条时长：3-8 秒（根据部位）"
  │  ├─ "提示：读条中受击将打断，需重新开始"
  │  └─ [开始] [取消] 两个按钮
  └─ 按 ESC 可取消关闭
```

### 交互规则

1. **部位选择**：单选（一次只能选 1 个部位）
2. **颜料选择**：单选（一次只能用 1 种颜料）
3. **图案选择**：单选
4. **开始**：
   - 检查选择是否完整（部位/颜料/图案都选了）
   - 若缺少某项，提示"请选择[部位/颜料/图案]"
   - 完整后发 `RequestSelfTattooEvent`
   - UIModule 自动关闭 Form，进入读条状态
5. **取消**：关闭 Form，返回战斗

### 界面尺寸与布局（参考）

- **整体**：1920×1080（全屏）或 1280×720（适配小屏）
- **部位按钮**：60×60px，间距 10px
- **颜料格子**：80×80px，间距 8px
- **图案缩略图**：100×100px，间距 10px
- **预览窗**：400×500px（竖长，突出部位）
- **按钮区**：标准 120px 高，间距 20px
